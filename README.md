# Alice Miner

Training miner for Alice Protocol.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python alice_miner_v2.py --ps-url https://ps.aliceprotocol.org --address YOUR_WALLET_ADDRESS
```

## Requirements
- Python 3.10+
- CUDA GPU (recommended) or MPS (Apple Silicon)
- 24GB+ VRAM for full training
